# 电商销售分析与策略优化系统

基于 Streamlit 的电商数据分析平台，支持完整的销售分析流程。

## 🚀 功能特性

- **数据预处理**：按论文要求生成6个标准化输出文件
- **多维特征分析**：品类×区域×利润热力图、客户-商品聚类
- **销售预测**：ARIMA+XGBoost混合模型预测未来趋势
- **运营优化**：ABC分类、价格敏感度分析、可落地策略

## 📊 系统界面

1. **数据预处理** - 生成论文要求的6个标准文件
2. **多维分析** - 交叉维度热力图和聚类分析
3. **销售预测** - 时间序列预测模型
4. **运营优化** - 智能策略推荐
